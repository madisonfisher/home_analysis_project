Zillow Housing Market Analysis

Welcome to our website analyzing Zillow Housing Market data. On this website you can find three analysis done with Machine learning as well as two other analysis to help you understand this new and abnormal housing market we have mid 2021. All the analyses can be seen below with a brief description. For additional information on how the data was process see the Methodology page.

Please note the analyses can be accessed by clicking on the image on each card or by using the navigational bar.

Inventory Analysis Map


On this map cities across the US are displayed and when a marker is selected a graph is revealed to show the mean and median time houses have been on the market starting in 2018. It additionally shows the data that was generated by machine learning as the predicted data.

Housing by Tier


On the graphs on this page Random Forest Regression was used to predict the average value of a home based on the value in the previous year. This was done three times: for homes in the top 35%, homes in the 65th to 35th percentile, and homes in the bottom 35%.

Rent Prices Tableau


In the Tableau graphs on this page there is an analysis of Average Annual Rents (2014-2020), Cities displayed by highest rent, and Average Rent Price vs. City Size.

Home Value Over Time


Zillow's Home Value Index data from 1996-2019 was split into before and after the housing market bubble. These data sets then were used to train a machine learning model to predict home values on June 30, 2021. The actual values and the predict values can be seen in the table on this page.

Sale vs. List Price


The rolling 6 month average is shown on this page's graph for the mean sale and list price from 2018 to mid 2021. Additionally the monthly data is plotted. The 2021 spike in price is clearly seen on this graph.
